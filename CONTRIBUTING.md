# Contributing to BioClusterQuant

Thank you for your interest in contributing to BioClusterQuant! We welcome feedback, bug reports, and code contributions from the community.

## Reporting Issues
If you find a bug or have a suggestion for a new feature, please open an Issue in this repository.
* **Bugs:** Please include a clear description of the error, screenshots if possible, and the type of data you were using.
* **Features:** Describe the goal of the new feature and how it would help your workflow.

## Contributing Code
If you want to fix a bug or add a feature yourself:
1.  **Fork** the repository to your own GitHub account.
2.  **Clone** the project to your machine.
3.  Create a new **Branch** for your changes.
4.  Make your changes and ensure the code is clean.
5.  **Run the Tests:** Before submitting, please verify the software still installs correctly by running:
    ```bash
    pytest tests/test_import.py
    ```
6.  Submit a **Pull Request (PR)** to the `main` branch.

## Questions?
If you have questions about the usage of the software, please check the README or open an Issue labeled "question".
